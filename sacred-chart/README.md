A Scrubs episode prediction game for Jean & Ed.

## How to Play
1. Set the season and episode you're watching
2. When someone makes a prediction, tap their name
3. Mark if they were correct or wrong
4. Track scores per episode, per season, and all-time

## Install on iPhone
1. Open the app in Safari
2. Tap the Share button
3. Select "Add to Home Screen"
4. Enjoy!
