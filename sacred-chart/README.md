A Scrubs episode prediction game.

## How to Play
1. Enter player names on first launch
2. Set the season and episode you're watching
3. When someone makes a prediction, mark if they were correct or wrong
4. Track scores per episode, per season, and all-time

## Install on iPhone
1. Open the app URL in Safari
2. Tap the Share button
3. Select "Add to Home Screen"
